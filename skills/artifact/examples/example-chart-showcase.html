ECharts Type Showcase

23 chart types organized by category. Each panel labels the chart type and its use case.

Press 1-7 for tabs. Esc to exit fullscreen.

Line Chart

Trends over time. Gradient area fill via LinearGradient. markLine shows a $150K target.

Bar Chart

Categorical comparison. markArea highlights Q4.

Stacked Area

Composition over time. stack: 'total' with areaStyle.

Line + Bar Combo

Dual-axis: revenue (bar, left) and growth rate (line, right).

Treemap

Hierarchical drill-down. Click a node to zoom in; breadcrumb to zoom out.

Funnel

Conversion pipeline. Data sorted descending by value.

Sankey

Flow between nodes. emphasis: { focus: 'adjacency' } highlights connected paths.

Scatter / Bubble

Size = probability, color = risk tier. symbolSize maps the third dimension.

Heatmap

2D density with visualMap color gradient. Data: [[x, y, value], ...]

Radar

Multi-dimensional comparison. Each indicator is an axis spoke.

Boxplot

Distribution: [min, Q1, median, Q3, max] per category.

Revenue Target

Gauge with red/amber/green threshold bands at 60/80/100.

Customer Satisfaction

Same gauge pattern, different KPI value.

System Uptime

Narrow range (95-100%). Custom band thresholds.

Candlestick + Volume

OHLC price action with volume bars. Dual y-axis. DataZoom slider below. Green = bullish, red = bearish.

Waterfall Chart

Bridge chart showing cumulative P&L. Stacked bars with transparent base create the stepping effect. Blue = subtotals, green = increases, red = decreases.

Calendar Heatmap

Daily contributions over a year. Uses ECharts calendar coordinate system with heatmap series. GitHub-style contribution graph.

Sunburst

Radial hierarchy. Click a sector to drill in; click center to zoom out.

Graph / Network

Force-directed layout. Drag nodes, pan/zoom. Hover highlights connected neighbors.

Parallel Coordinates

Each vertical axis is a dimension. Each line is a server config. Price inverted.

ThemeRiver

Centered streamgraph. Width shows category volume over time.

Polar Bar (Nightingale Rose)

Radial bar chart for cyclical data. Monthly revenue around a circle.

Bullet Chart

Actual vs target with qualitative range bands. Compact KPI comparison.

Lollipop Chart

Dot-on-stem ranking. Higher data-ink ratio than bar charts.

Small Multiples

Grid of mini-charts, one per service. Same scale for direct comparison.
