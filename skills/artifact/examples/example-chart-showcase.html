ECharts Type Showcase

All 13 chart types organized by category. Each panel labels the chart type and its use case.

Press 1-5 to switch tabs. Esc to exit fullscreen.

Line Chart

Trends over time. markLine shows a $150K target threshold.

Bar Chart

Categorical comparison. markArea highlights Q4.

Stacked Area

Composition over time. stack: 'total' with areaStyle.

Line + Bar Combo

Dual-axis: revenue (bar, left) and growth rate (line, right).

Donut / Pie

Proportions. radius: ['40%', '70%'] makes the donut hole.

Treemap

Hierarchical drill-down. Click a node to zoom in; breadcrumb to zoom out.

Funnel

Conversion pipeline. Data sorted descending by value.

Sankey

Flow between nodes. emphasis: { focus: 'adjacency' } highlights connected paths.

Scatter / Bubble

Size = probability, color = risk tier. symbolSize maps the third dimension.

Heatmap

2D density with visualMap color gradient. Data: [[x, y, value], ...]

Radar

Multi-dimensional comparison. Each indicator is an axis spoke.

Boxplot

Distribution: [min, Q1, median, Q3, max] per category.

Revenue Target

Gauge with red/amber/green threshold bands at 60/80/100.

Customer Satisfaction

Same gauge pattern, different KPI value.

System Uptime

Narrow range (95-100%). Custom band thresholds.

Candlestick + Volume

OHLC price action with volume bars. Dual y-axis. DataZoom slider below. Green = bullish (close > open), red = bearish.
